# 知乎文章与毛玻璃主页设计

## 目标

将知乎个人页当前展示的 11 篇文章完整、准确地同步到个人主页，并把现有静态主页升级为清晰、克制、兼顾可读性的毛玻璃界面。

## 内容设计

- “随笔”区按知乎当前时间顺序展示全部 11 篇文章。
- 每张文章卡片包含知乎标识、文章标题、简短摘要和“阅读全文”链接。
- 文章标题与链接以 2026-06-07 登录态下的知乎文章页为准。
- 区块标题旁提供“查看全部知乎文章”入口。
- 静态数据直接写入 HTML，避免 GitHub Pages 上的跨域、反爬和登录态依赖。

## 视觉设计

- 使用浅色冷灰背景、青绿色主色与少量珊瑚色强调，避免单一色相。
- 导航、Hero、内容卡片使用半透明背景、背景模糊、内高光和轻阴影形成玻璃层次。
- 不使用漂浮光球；背景改为固定的细网格与大面积柔和色带，让玻璃模糊有可见参照但不喧宾夺主。
- Hero 保留个人头像与核心履历，减少装饰性组件的体积。
- 文章卡片使用紧凑网格，最新三篇添加“NEW”标记。
- 链接和按钮保留清晰的键盘焦点状态。

## 响应式与动效

- 桌面端使用双栏 Hero 和三栏文章网格。
- 平板端降为两栏；移动端降为单栏并使用可横向滚动的导航。
- 滚动入场动画只播放一次。
- `prefers-reduced-motion` 下关闭非必要动画。
- 不支持 `backdrop-filter` 时保留足够不透明度，确保文字可读。

## 验证

- Node 静态测试验证 11 篇文章、链接映射、知乎入口及毛玻璃关键样式。
- 本地静态服务器验证无资源加载错误。
- 浏览器分别以桌面和移动视口检查布局、文字溢出、导航和链接。
